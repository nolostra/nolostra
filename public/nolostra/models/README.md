# 3D Models Directory

Place your downloaded Porsche 3D model file here.

## Instructions

1. Download the GLB file from Sketchfab:
   - https://sketchfab.com/3d-models/free-1975-porsche-911-930-turbo-8568d9d14a994b9cae59499f0dbed21e

2. Rename it to `porsche.glb` and place it in this directory.

3. The final path should be: `public/nolostra/models/porsche.glb`

## File Format

- **Preferred**: `.glb` (single file, optimized)
- **Alternative**: `.gltf` (if GLB is not available)
